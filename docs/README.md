# NovaLMS

NovaLMS es una plataforma de aprendizaje modular inspirada en Moodle 4.x construida con un stack 100% JavaScript/TypeScript. Este documento ofrece una visión general del repositorio y enlaces a la documentación especializada.

## Estado actual

> **Nota:** Este repositorio contiene la primera iteración enfocada en la infraestructura del monorepo, lineamientos de calidad y una API mínima de referencia. Las funcionalidades complejas descritas en los requisitos se planifican y documentan en `docs/DECISIONES.md` junto con el roadmap incremental.
